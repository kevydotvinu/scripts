Windows OS installation using WinPE

Step 1: Get IP: wpeinit

Step 2: Confirm IP is assigned: ipconfig

Step 3: Map share directory: net use I: \\172.16.1.87\ISO\Win10 /user:user pass

Step 4: Start installation: I:\setup.exe


How to add Windows ISO image to PXE server

Step 1: Goto \\172.16.1.87\ISO

Step 2: Create a folder to copy bootable DVD files. Name should be meaningful like Win10

Step 3: Copy files inside the bootable DVD and paste it directly to the folder we created on step two

Step 4: SSH to user@172.16.1.87 and restart PXE service: sudo systemctl restart pxe Put password: password
